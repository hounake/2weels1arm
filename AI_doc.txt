	-------	FILE NAME --------
  ROOT   =>    SEPARATOR    =>    NUMBER GENERATION
(string)	   _		(int)
	--------------------------


Create => start AlgoGen from the begining:
	- total number of generation
	- size of a generation
	- file name for saving the log (root)

Reload => continue AlgoGen from a FILENAME (size is written in the file we load)
	- total number of generation 
	- file name to load (root + separator + numberGeneration)

ReplayRank => replay the Xth best score robot
	- file name to load (root + separator + numberGeneration)
	- rank of the robot we want to show

MakeCSV => group the civilisation's logs on an unique csv file
	- file name (root)
